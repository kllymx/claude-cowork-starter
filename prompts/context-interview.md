# Context Interview Prompt

**Paste this into a new Cowork session to have Claude interview you and build your context files automatically.**

This is the highest-leverage thing you can do in your first 20 minutes. Instead of filling out templates manually, Claude asks you targeted questions and builds the files based on your answers.

---

## The Prompt

```
You are going to help me set up my Claude Cowork workspace so that every future session starts with full context about who I am, what I do, and how I work.

You're going to interview me in phases. Ask me questions one at a time, then build the files based on my answers. Don't rush. Don't assume. Ask before you build.

Phase 1: About Me
Interview me to create an about-me.md file in /context. Ask about:
- My role, company, and what I actually do day-to-day
- My current priorities and what success looks like
- My industry, key terminology, and tools I use
- 1-2 examples of work I'm proud of (ask me to paste them)

Create the file, show it to me, and get my approval before moving on.

Phase 2: Brand Voice
Interview me to create a brand-voice.md file in /context. Ask about:
- How I naturally communicate (tone, style, energy)
- Phrases I use and phrases I'd never use
- How my tone shifts by context (clients vs team vs public)
- Ask me to paste 2-3 samples of my actual writing

Create the file with voice rules, tone by context, dos and don'ts. Get approval.

Phase 3: Working Preferences
Interview me to create a working-preferences.md file in /context. Ask about:
- What I want Claude to help with daily
- How I want Claude to communicate with me
- My biggest workflow pain points
- Output format preferences
- Quality standards and non-negotiables

Create the file. Get approval.

Phase 4: Validation
After all three files are created, read them back and summarize:
1. What you now know about me
2. How I prefer to work
3. What guardrails you should follow

Ask me if anything is off. Fix anything that needs fixing.

Rules:
- Interview me one phase at a time
- Ask one question at a time (max 12 questions total across all phases)
- Show each file before saving
- If unsure about anything, ask — don't assume
- Keep files concise and scannable
- Save everything to /context
```

---

## What You'll Get

After ~15 minutes of conversation, you'll have three files that make every future Cowork session start with full context about you. No more generic output. No more re-explaining yourself every session.

These files compound over time — update them whenever Claude misses the mark.
